# 🎧 Audio Player Project

##  Overview

This project is a simple **Audio Player application** built using C++.
It provides basic functionality for playing audio files along with a graphical user interface.

---

## Project Structure

```
.
├── Main.cpp
├── MainComponent.cpp
├── MainComponent.h
├── PlayerAudio.cpp
├── PlayerAudio.h
├── PlayerGUI.cpp
├── PlayerGUI.h
```

### 🔹 File Descriptions

* **Main.cpp**
  Entry point of the application. 

* **MainComponent.cpp / MainComponent.h**
 .

* **PlayerAudio.cpp / PlayerAudio.h**
  

* **PlayerGUI.cpp / PlayerGUI.h**


---

##  Features

* ▶️ Play audio files
* ⏸️ Pause functionality
* ⏹️ Stop playback
* 🎛️ Simple GUI interface
